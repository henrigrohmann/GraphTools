# フロント・バック結合テスト手順（Codespaces向け）

Codespacesの制限により、ターミナル操作なしで結合テストを進める手順です。

## 環境

- フロント: `http://localhost:8002` (UI確認)
- バック: `http://localhost:8005` (API)

## 起動手順

### 方法1: VS Code タスク実行（推奨）

1. VS Code の **Terminal** → **Run Task** を開く
2. 以下を順に実行:
   - **"Start FastAPI (uvicorn)"** → バックエンド (8005) 起動
   - **"Start Frontend Static Server"** → フロント (8002) 起動

3. 起動完了後、ブラウザで開く:
   - メイン UI: `http://localhost:8002`
   - API テスター: `http://localhost:8002/api_tester.html`

### 方法2: launch.json デバッグ設定

VS Code の Run and Debug (Ctrl+Shift+D) から実行可能（※別途セットアップ必要）

## 結合テスト フロー

### 1️⃣ ヘルスチェック

メイン UI (`http://localhost:8002`) で:
- **[ヘルスチェック]** ボタンをクリック
- ログパネルに `HEALTH OK ...` と表示されれば、フロント↔バック疎通OK

### 2️⃣ 散布図表示テスト

メイン UI で以下ボタンを順にクリック:

| ボタン | テスト内容 | 期待 |
|--------|---------|------|
| **[生データ]** | /raw → /scatter?mode=raw | 散布図が表示される |
| **[ランダム]** | /random → /scatter?mode=random | ランダム配置の散布図 |
| **[クラスタリング]** | /cluster → /scatter?mode=cluster | クラスタリング後の散布図 |
| **[濃い意見]** | /dense → /scatter?mode=dense | 密度フィルタ後の散布図 |

### 3️⃣ 詳細表示テスト

散布図の点をクリック → 右パネルの「詳細」タブに ID と内容が表示される

### 4️⃣ 階層ビュー テスト

散布図読み込み後、右パネルの「階層ビュー」タブを開く → クラスタ単位の階層が表示される

### 5️⃣ API スモークテスト（オプション）

細かさが必要な場合:

1. API テスター (`http://localhost:8002/api_tester.html`) を開く
2. **[Run All]** をクリック
3. 全テスト結果がテーブルに表示

## トラブルシューティング

### ❌ "HEALTH NG" または接続エラー

1. **バックエンド起動状態を確認**
   - VS Code 下部のターミナルウィンドウに uwvicorn ログが表示されているか確認
   - ポート 8005 が既に使われていないか確認

2. **ファイアウォール/Codespaces設定**
   - ポート 8005 が公開設定になっているか確認（ポート欄で確認）

3. **再起動**
   - 該当タスクを「停止」して再度「実行」

### ❌ 散布図が出ない（フレームは出る）

1. **ログパネルを確認**: 下部の緑文字ログにエラーメッセージが出ているか
2. **API Base が正しいか**: 上部の「API Base:」表示を確認
   - Codespaces: `https://xxxxx-8005.app.github.dev`
   - ローカル: `http://127.0.0.1:8005`
   - 異なる場合は URL に `?apiBase=...` を追加

3. **パイプラインが実行されているか**: [ヘルスチェック] を実行して jobs 数が増えているか確認

### ❌ フロント (8002) に接続できない

1. VS Code ポート欄で 8002 が表示されているか確認
2. 表示されていない場合、タスクがハング中の可能性
   - ターミナルの該当タスクを停止
   - 再度起動

## ファイル構成

- `index.html` - メイン UI
- `scatter.js` - 散布図・詳細・階層の統合スクリプト
- `api_tester.html` - API テスター画面
- `api_tester.js` - API テスター実装
- `backend/server.py` - FastAPI バックエンド
- `scripts/smoke_api.py` - スモークテスト (CLI)

## 既知の制限

- Codespacesではターミナル進行中にハングすることがあり、その場合は新しいタスク実行に切り替え
- ブラウザキャッシュの影響で古い API ベースが残ることがあるため、DevTools でキャッシュ無効化 ON 推奨

---

**最終確認**: メイン UI で [生データ] → [クラスタリング] の両方で散布図が表示されれば、結合テスト成功です！
